# Servlet
학습목표
- 자바 웹 어플리케이션의 구조를 이해한다.
- 서브릿에 대하여 이해한다.

### 자바 웹 어플리케이션(Java Web Application)
- WAS에 설치되어 동작하는 어플리케이션
- 자바 웹 어플리케이션에는 HTML,CSS,이미지,자바로 작성된 클래스, 각종 설정 파일 등이 포함된다.
- 블로그, 쇼핑몰과 같은 것들이 web app이다.
